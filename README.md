# Kaede Nakamura Portfolio

## プロジェクト概要

中村楓さんのポートフォリオサイトです。理学療法士からITエンジニアへのキャリアチェンジを目指す方のプロフェッショナルなポートフォリオを表現しています。

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: モダンなスタイリングとアニメーション
- **JavaScript**: インタラクティブな機能実装
- **Tailwind CSS**: ユーティリティファーストなCSSフレームワーク
- **Lucide Icons**: 美しいアイコンセット

## 主な機能

### 1. スクロールアニメーション
- セクションごとのフェードイン効果
- スムーズなスクロール移動
- インターセクションオブザーバーによる最適化

### 2. インタラクティブなキャリアセクション
- アコーディオン形式の経歴表示
- ホバーエフェクトと展開アニメーション
- 動的なコンテンツ生成

### 3. レスポンシブデザイン
- モバイルファーストの設計
- ブレークポイントに応じた最適表示
- タッチデバイス対応

### 4. アクセシビリティ対応
- セマンティックなHTML構造
- キーボードナビゲーション
- スクリーンリーダー対応

## カスタマイズ方法

### 画像の変更
`images/`フォルダーに新しい画像を配置し、`index.html`の画像パスを更新してください。

### 色の変更
`css/style.css`のCSS変数や色コードを変更することで、サイト全体の色を変更できます。

### コンテンツの更新
`index.html`の該当セクションを編集することで、コンテンツを更新できます。

### 経歴データの更新
`js/main.js`の`careerData`配列を編集することで、キャリアセクションの内容を更新できます。

## デプロイ方法

1. このリポジトリをクローンまたはダウンロード
2. 必要に応じてコンテンツをカスタマイズ
3. GitHub Pages、Netlify、Vercelなどの静的サイトホスティングサービスにデプロイ
