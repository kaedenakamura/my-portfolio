🌿 技術の先に、体温の通った解決を。
こんにちは、中村 楓です。 これまで理学療法士として身体を支え、営業として心を動かす仕事に携わってきました。 
今は「技術」という新しい道具を手に、誰かの日常を少しだけ温かく、便利にするための学びを続けています。

私が大切にしていること
効率や速さだけではなく、そのツールを使う「人」の体温を感じられる開発を志しています。

「なぜ？」の先にある笑顔のために
理学療法士時代、もっと正確に患者さんの状態を知りたくて、自費でポータブルエコーを購入しました。
それは技術への興味以上に、「目の前の人を何とかしてあげたい」という願いからでした。この探究心は、今のIT学習においても、効率化のためだけではなく、使う人が笑顔になれたり喜んでもらえる。そういう気持ちで作りたいです。

「寄り添い」をシステムにする
サッカーの副キャプテン時代や人材営業の経験を通じて学んだのは、相手の言葉にならない悩みを聞き取ることの大切さです。システムも同じだと考えています。使う人が「わかってもらえている」と感じるような、温かい設計を目指しています。

素直に、ひたむきに 
新しい環境でゼロから学ぶことは、時に勇気がいります。しかし、周囲の方々の知恵を素直に吸収し、一歩ずつ着実に進むことで、いつか誰かを支えられる大きな力になると信じています。

想いを込めて作成させていただきました。お忙しい所恐縮ですが、ご閲覧頂けたら幸いです。
