🌿 技術の先に、体温の通った解決を。

初めまして、中村 楓です。ご閲覧頂きありがとうございます。 理学療法士として身体を支え、営業として心を動かす仕事を経て、今は「技術」という新しい道具を使って、誰かの日常を少しだけ温かく、便利にするための学びを始めました

私が大切にしていることは効率や速さだけではなく、そのツールを使う「人」の体温を開発することを志しています。

「なぜ？」の先にある笑顔のために 理学療法士時代、もっと正確に患者さんの状態をたくて、自費でポータブルエコーを購入しました。それは技術への興味以上に、「目の前の人をそうしてあげたい」という願いからでした。この探究心は、今のIT学習においても、効率化の為だけではなく、使う人が笑ったり喜んだり、そう感じてほしいと願う気持ちで、動きを待っています

「寄り添い」をシステムにするサッカーの副キャプテン時代や人材営業の経験を学んだのは、相手の言葉にならない悩みを聞くことの大切さです。システムも同じだと考えています。

素直に、ひたむきに新しい環境でゼロから学ぶことは、時に勇気がいります。しかし、周囲の人々の知恵を素直に吸収し、一歩ずつ着実に進むことで、いつか誰かを支えられる大きな力になると信じています。

今、磨いているツール（技術スキル）「現場の負」を温かな解決に変えるために、以下の技術を学んでいます。 フロントエンド：心地よい手触りを作る（HTML5、CSS3、JavaScript React Recharts） バックエンド：信頼の土台を作る（Node.js、Express、hono　Firebase） データベース：大切な情報を預かる（PostgreSQL、Prisma Firestore） インフラ：安定した居場所を作る（Docker、Docker Compose）品質への誠実さ：確かな安心を届ける
